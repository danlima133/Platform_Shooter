# Platform Shooter
Um simples jogo de platforma atirador, para praticar Godot 3 e relembrar alguns conceitos importantes.

## Logs

- Finalizar alguns testes unitarios

Mais detalhes dos testes podem ser encontrados na pasta **reports**
